## Phase 6: Rizz System (Diva Foundation)

### 016 - Rizz Meter
**Severity:** med

Track flower's attention/affection.

**Acceptance Criteria:**
- Each FlowerHead has rizz value (0-100)
- Small meter displayed above each head
- Rizz decays at 5/second
- Low rizz (<30): head moves toward nearest bee
- High rizz (>70): head moves in lazy circles (predictable)
